# FFmpeg Assistant GUI

一个 **简洁、现代、支持 GPU 加速** 的音视频格式转换工具，基于 **FFmpeg + PyQt5 + Fluent 设计** 构建。

---

## 特性

- 支持 **30+ 主流音视频格式**（MP4, MKV, AVI, MP3, FLAC, GIF 等）
- **NVIDIA/AMD GPU 硬件加速**（`h264_nvenc`）
- 实时 **进度条 + 详细日志**
- 自定义 **帧率、分辨率**
- **Fluent 2 现代界面**（`qfluentwidgets`）
- 支持 **批量处理**（后续版本）

---

## 快速开始

### 方式 1：直接运行（推荐）

1. 下载最新版 [`FFmpegAssistant.exe`](https://github.com/Dr1mH4X/FFmpegAssistant/releases)
2. **双击运行**
3. 选择文件 → 设置格式 → 点击 **开始转换**
